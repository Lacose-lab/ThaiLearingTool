# Handover — 2026-04-19

## What this app is

**Kru Noi** — a personal Thai vocabulary learning PWA for a German male living in Bangkok, intermediate level, classes twice a week. Built entirely in vanilla JS ES modules, deployed on GitHub Pages, no build step.

**Live URL:** https://lacose-lab.github.io/ThaiLearingTool/
**Repo:** https://github.com/Lacose-lab/ThaiLearingTool
**Local:** `D:/AI_Project/thai_learning_app`

---

## Stack

| Layer | What |
|-------|------|
| Frontend | Vanilla JS ES modules, zero dependencies |
| Hosting | GitHub Pages (auto-deploys on push to `main`) |
| Vocab source | Google Sheets CSV — syncs on every app load |
| AI | Anthropic claude-haiku-4-5 via Cloudflare Worker proxy |
| Offline | PWA — manifest.json + sw.js (currently v7) |
| SRS | SM-2 spaced repetition, progress in localStorage |

---

## Architecture

```
index.html
└── src/js/app.js          — bootstrap, tab routing, SW registration, reminders
    ├── home.js             — streak, due count, weak words widget, Quick 5
    ├── flashcards.js       — SRS cards, category filter, quick5/weak modes
    ├── quiz.js             — multiple choice, wrong-answer pause + "Got it"
    ├── typing.js           — type the Thai
    ├── tutor.js            — Thai Teacher: drill mode + AI chat
    ├── settings.js         — Profile: reminder, vocab sync, progress
    ├── storage.js          — localStorage wrapper (progress, streak, etc.)
    ├── srs.js              — SM-2 algorithm, getDueWords, updateCard
    ├── tts.js              — Web Speech API th-TH
    ├── romanize.js         — RTGS-approximation syllable-level romanization
    └── sheets.js           — Google Sheets CSV live sync
```

**Cloudflare Worker:** `cloudflare-worker.js` (repo root)
- URL: `https://kru-noi-proxy.lindnermanuel1992.workers.dev`
- Proxies POST to `api.anthropic.com/v1/messages`
- API key stored as Worker secret `ANTHROPIC_API_KEY`
- Model hardlocked to `claude-haiku-4-5` server-side

## Key files & their current state

### `src/js/tutor.js`
The Thai Teacher tab. Two modes:

**Drill mode** (local, no AI, free):
- ⚡ Drill me button → picks weak/due words → shows Thai script large
- 4 multiple choice English options (tap to answer)
- "Show pronunciation" reveals romanization before answering
- "🔊 Hear it" speaks the word
- After answer: romanization always auto-shows, feedback shown
- Wrong answers pause 1.8s, correct 0.9s
- After 10 words: score + "Drill again"

**AI chat** (uses Cloudflare Worker):
- 💬 Let's talk → free conversation practice
- 📖 Recap today → weak word summary from AI
- Text input at bottom for free questions
- Streaming SSE response, speak button on each reply
- System prompt includes full 488-word vocab + weak words, cached with `cache_control: ephemeral`

### `src/js/storage.js`
Key localStorage keys:
```
yt_research_progress    — SRS progress per word {interval, easeFactor, nextReview, reviews, failures}
yt_research_streak      — day streak count
yt_research_last_seen   — last date streak was updated
yt_reminder_time        — HH:MM for daily notification (default 20:00)
yt_last_practice        — date string of last practice
kru_noi_worker_url      — Cloudflare Worker URL (has hardcoded default)
```

### `sw.js`
Cache name: `kru-noi-v7`. Bump version every deploy that changes JS files so PWA users get fresh code.

## How to deploy changes

```bash
# Edit files, then:
git add <files>
git commit -m "description"
git push origin main
# GitHub Pages auto-deploys in ~1 min
# Also bump sw.js cache version (kru-noi-vN → kru-noi-vN+1) when changing JS
```

## Cloudflare Worker — how to update

1. dash.cloudflare.com → Workers & Pages → kru-noi-proxy → Edit code
2. Paste updated `cloudflare-worker.js`
3. Deploy
4. Secrets: Settings → Variables and Secrets → `ANTHROPIC_API_KEY`
